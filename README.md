# 🪔 Diwali Sales Analysis Dashboard

## 📌 Overview

This project presents an end-to-end **data analysis and visualization pipeline** for Diwali sales data. The objective is to extract meaningful business insights related to customer behavior, product performance, and regional trends.

The workflow includes:

* Data cleaning using **Python (Pandas)**
* Exploratory analysis in **Jupyter Notebook**
* Interactive dashboard creation in **Power BI**

## 🎯 Objectives

* Analyze customer demographics (age, gender, occupation)
* Identify top-performing product categories
* Determine high-demand states
* Understand purchasing patterns during Diwali sales

---

## 🛠️ Tools & Technologies Used

* **Python** → Data cleaning & preprocessing
* **Pandas** → Data manipulation
* **Jupyter Notebook** → Data analysis workflow
* **Power BI** → Dashboard & visualization
* **Kaggle** → Dataset source

---

## 🔍 Key Insights

* 📈 **Top Buying Age Group**: Customers aged **25–35** contribute the highest number of orders
* 🛍️ **Best-Selling Categories**: Clothing & Apparel and Electronics dominate sales
* 🌍 **Top States**: Majority of orders come from a few key states like Uttar Pradesh and Maharashtra
* 👨‍💼 **Customer Segments**: IT and Healthcare professionals form a significant portion of buyers
* 👥 **Gender Trends**: Female customers contribute more to total orders compared to males
